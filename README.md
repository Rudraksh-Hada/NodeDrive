## ⚠️ Deployment Note

This project is deployed using the **free tiers** of Vercel and Render.

Because the backend is hosted on Render's free plan, the server may go into **sleep mode after inactivity**. When this happens, the first request can take **30–60 seconds to wake the server**. After the initial wake-up, the application will respond normally.

If the app seems slow on the first load, please wait a few seconds for the backend service to start.

Thank you for your patience!
